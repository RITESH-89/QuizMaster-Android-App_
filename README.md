# 🧠 QuizMaster – Android Quiz App

## 📌 Overview

QuizMaster is a simple Android Quiz Application built using Java and XML in Android Studio.

The app allows users to answer True/False questions, navigate between questions, and view their final score at the end of the quiz.

This project demonstrates Android UI design, event handling, and basic object-oriented programming concepts.

---

## ✨ Features

- ✅ True / False Quiz Questions
- ⏭ Navigate to Next Question
- ⏮ Navigate to Previous Question
- 🖼 Question-based Image Display
- 📊 Final Score Display
- 🔔 Toast Messages for Feedback
- 📱 Clean and Simple UI

## ⚙ How It Works

1. Questions are stored in a Question class array.
2. Each question contains:
   - A string resource
   - A boolean answer
3. User selects True or False.
4. App checks correctness using logic.
5. Score is calculated and displayed at the end.
